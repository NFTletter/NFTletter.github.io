	    Letter is an ERC-721 Smart Contract that allows for the creation of text-based NFTs, meant for written artistic expression.
      

      Letter allows for art pieces such as poems, novels, manifestos and source code to be minted as Non-Fungible Tokens into the Ethereum blockchain.
      

      

      Each Non-Fungible Token represents a Page with the following fields:
      

	    


      

      

      
        

          Field

          Description
          Encoding
          Optional?
        

      
      
        

          Id
          Page Id
          HEX
          no
        

        

          Nonce
          Page counter
          HEX
          no
        

        

          Title
          Title of the Page
          Unicode
          yes
        

        

          Body
          Actual contents of the Page
          Unicode
          no
        

        

          Rubric
          Author signature
          Unicode
          yes
        

        

          Parent
          Id of parent Page
          HEX
          yes
        

      
      


    

    Unicode allows for the widest possible range of digital text-based expression.
