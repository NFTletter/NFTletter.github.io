	    Letter is an ERC-1155 Smart Contract that allows for the creation of text-based NFTs, meant for written artistic expression.
      

      Letter allows for art pieces such as poems, novels, manifestos and source code to be minted as Non-Fungible Tokens into the Ethereum blockchain.
      

      

      Each Non-Fungible Token represents a Page, encoded as a .json file to be pinned on IPFS:
      

	    


      
    

    
      

        Field

        Description
        Encoding
        Optional?
      

    
    
      

        Id
        Page/Token Id
        uint256
        no
      

      

        Nonce
        Page number
        uint256
        no
      

      

        Cover
        Page cover
        ipfs://cid for .png or .svg
        yes
      

      

        Title
        Title of the Page
        ipfs://cid for .txt (UTF-8)
        yes
      

      

        Body
        Actual contents of the Page
        ipfs://cid for .txt (UTF-8)
        no
      

      

        Rubric
        Author signature
        ipfs://cid for .txt (UTF-8)
        yes
      

      

        Parent
        Id of parent Page/Token
        uint256
        yes
      

    
    


    

    NFT Pages are minted, burned and traded against the LETT token.
    

    .json, .txt, .png and .svg files are pinned to IPFS over Pinata.
